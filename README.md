# Geometry Analysis Tool

Dieses Projekt berechnet den Umfang von geometrischen Formen basierend auf Punkt-Koordinaten.

## Features
- Berechnung des Perimeters beliebiger Polygone.
- Rein in Standard-Java geschrieben (keine externen Uni-Libraries).
- Fokus auf Objektorientierung (OOP).
